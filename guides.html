Best path

What should a brand-new person do first?

1

Choose a wallet

Pick a wallet built for beginners and learn how to back it up correctly.

2

Buy a small amount

Start small. Learn the flow before you commit larger amounts.

3

Send a test payment

Practice sending and receiving so you understand how Litecoin actually works.

4

Spend or accept it

Use it in real life or help a business start accepting it.
